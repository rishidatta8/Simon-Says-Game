How To Play

This is The Simon Game. It tests your ability to remember patterns and helps improving it.


  1. The game starts with one button. You need to remember the color of the button that blinked.
  2. After that click on the button that blinked. If it is correct, the game will proceed and you'll enter the next level.
  3. If you click the wrong colored button, the game will end.
  4. Each time you press the correct button, a new colored button is added to the pattern.
  5. You need to remember the corerect order and click the buttons according to that.
  6. Good Luck with the game!
